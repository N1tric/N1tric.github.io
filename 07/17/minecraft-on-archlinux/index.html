Run Minecraft on Archlinux

To run Minecraft on Archlinux, you need JRE(Java Runtime Environment), JavaFX, and a Minecraft Launcher.

Install Java Environment

¶Open-source implement

For most users who dont play with java programming, the open-source version is enough for gaming.
Using package: jre8-openjdk, java-openjfx
pacman -S jre-openjdk java-openjfx

¶Oracle implement

Generally, closed-source implement often has better performance than open-source.
Using package: jre8(in aur)
yay -S jre8

¶Change environment between installed jre

Use archlinux-java

Install Minecraft Launcher

Using package: hmcl
pacman -S hmcl

¶Install GTK2

If you dont get GTK2 installed, then when you try to starting hmcl, it will output an exception, such like:

Just simply install the package gtk2 can solve the problem.
